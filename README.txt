# Portfólio - Laís Müller

Este é meu portfólio pessoal! Feito especialmente para o Projeto Pessoal do Módulo 1, no curso da Resília.
Aqui compartilho informações sobre mim, meus hobbies e um breve formulário de contato.


## Tecnologias utilizadas

- HTML5
- CSS3


## Instalação

1. Faça o clone deste repositório para o seu ambiente local.
2. Navegue até o diretório do projeto.
3. Abra o arquivo `index.html` em um navegador web.

## Link do site GitHub e Repositório

https://laismullerrr.github.io/Portfolio/index.html
https://github.com/laismullerrr/Portfolio


## Estrutura do projeto

O projeto é estruturado da seguinte forma:

- `index.html`: Página inicial do portfólio e breve apresentação.
- `sobre.html`: Página com informações sobre mim.
- `hobbies.html`: Página com meus hobbies e interesses.
- `contato.html`: Página com o formulário de contato.


## Diário do Projeto:

Realizar esse portfolio foi um grande desafio. Estava parada na programação e não me recordava de muitas coisas em HTML/CSS, mas fiquei satistfeita com meu resultado.
Tive muita ajuda de colegas durante o processo, além de me inspirar em projetos vistos. Confesso também que o ChatGPT foi um grande aliado nessa jornada. 
Comecei de uma forma totalmente diferente (acabei perdendo arquivos, que deletei por não achar importantes) e aprendi que tudo bem errar. Me comparei com os outros projetos e decidi mudar a maioria do escopo do projeto, aos poucos fui acertando meu caminho.
Mas fiquei satisfeita com o resultado, por ora! Quero trabalhar mais nele assim que adquirir mais conhecimento, futuramente.
